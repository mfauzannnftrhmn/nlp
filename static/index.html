Tentang SentiVision

Sistem analisis sentimen berbasis Machine Learning yang dibangun untuk mengklasifikasikan ulasan produk Amazon sebagai positif atau negatif.

🧠

Multinomial Naive Bayes

Algoritma probabilistik yang sangat efektif untuk klasifikasi teks. Menggunakan teorema Bayes dengan asumsi independensi fitur.

📊

TF-IDF Vectorizer

Term Frequency-Inverse Document Frequency mengubah teks menjadi vektor numerik, mempertimbangkan kepentingan setiap kata dalam konteks dokumen.

📈

Confusion Matrix

Matriks evaluasi yang menampilkan True Positive, True Negative, False Positive, dan False Negative untuk mengukur performa klasifikasi.

🗃️

Amazon Dataset

Dataset berisi 1.000 ulasan produk elektronik dari Amazon, dilabeli secara manual sebagai sentimen positif (1) atau negatif (0).
